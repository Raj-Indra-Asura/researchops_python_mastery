# Sample Papers

This directory contains small PDF files for manual testing.

To test the `scan` and `ingest` commands, add a few PDF files here:

```bash
researchops scan ./examples/sample_papers
researchops ingest ./examples/sample_papers
```

## Adding test PDFs

You can add any small PDF here. Some free sources:
- [arXiv](https://arxiv.org) — search for short papers
- [PapersWithCode](https://paperswithcode.com) — ML papers

The `ingest` command (Week 6) will extract text and store them.
