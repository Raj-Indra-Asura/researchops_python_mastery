# Exercises - Week 17 RAG Assistant

## Warm-up exercises
1. Build a prompt string from a question and two context snippets.
2. Add citation labels like `[1]` and `[2]` to retrieved chunks.
3. Return a fallback answer when no context is available.
4. Write a small answer object containing text and citations.

## Project exercises
1. Implement a prompt builder for grounded Q&A.
2. Connect retrieval results to an assistant service.
3. Return answers with citation metadata.
4. Write tests for prompt structure, empty-context behavior, and grounded-answer flow.

## Stretch exercises
1. Add a confidence or evidence count field to answers.
2. Combine keyword and semantic retrieval before prompting.
3. Add a command or API route for interactive questioning.

## Writing questions
1. Why are citations essential in a RAG system?
2. What should happen when retrieval is weak?
3. Where can hallucinations still appear even with grounding?
4. How would you evaluate answer usefulness?
